# Example

> Create a TypeScript project in seconds

## Installation:

```bash
$ cd my-project
$ git clone https://github.com/CitRUSprod/example.git . && rm -rf .git README.md && git init && yarn && yarn sync-project-name && yarn
```

## Usage

### Development build

```bash
$ yarn dev
$ yarn dev:watch
$ yarn dev:watch:nodemon
```

### Production build

```bash
$ yarn build
```

### Create documentation

```bash
$ yarn docs
```

### Create a new package

```bash
$ yarn create-package
```

### Sync project name with folder name (when changing the folder name)

```bash
$ yarn sync-project-name
```

### Testing

```bash
$ yarn test
```

### Create release

```bash
$ yarn release
```
